Re-introduction
to Progressive Web App

PWA are Web Apps that make use of latest Web capabilities with progressive enhancement as backbone model.

They features:

  • Connectivity independent
  • App-like
  • Fresh
  • Safe
  • Discoverable
  • Re-engageable
  • Installable
  • Linkable

Minimum technical specs to be called PWA

  • https: safe, fresh
  • manifest.json: installable, discoverable,
  • SPA: app-like, linkable
  • Service worker: connectivity independent, re-engageable

Web capabilities we'll explore:

  • Installation and fullscreen mode
  • Multi-tasking
  • Working offline
  • Web Share API
  • Payment Request API
  • Credential Management API
  • Web Authentication API
  • Push notification
  • Background sync
